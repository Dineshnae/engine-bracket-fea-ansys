## Summary

Developed and validated a structural simulation model of an aircraft engine bracket, evaluating response across 6+ load cases including vibration and thermal conditions. Applied FEA-based V&V comparing simulation results against reference behaviour and aerospace loading standards.

## Key Results

| Metric | Value |
|---|---|
| Mass reduction via topology optimisation | **~12–15%** |
| Structural integrity | ✅ Maintained within aerospace loading limits |

## Tools

`ANSYS Mechanical` · `ANSYS Topology Optimisation` · `CATIA V5` (geometry import)

## Methodology

1. **Geometry preparation** — engine bracket CAD model imported from CATIA V5
2. **Meshing strategy** — refined mesh at fastener holes and stress concentration zones
3. **Load case definition** — static aerospace loads, vibration spectra, thermal conditions
4. **Linear + non-linear static FEA**
5. **Topology optimisation** — mass minimisation subject to stress and displacement constraints
6. **V&V** — simulation results benchmarked against reference behaviour and engineering standards
7. **Formal technical report** documenting assumptions, methodology, V&V approach, and design recommendations
